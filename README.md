# ST-TRPG-Tool

为 SillyTavern 提供 TRPG（桌面角色扮演游戏）工具箱的 UI 扩展。

## 功能特性

- 🎲 **骰子系统**：支持多种骰子类型（d6, d20, 2d6, 3d6等）
- 🗺️ **地图功能**：（计划中）
- 📚 **规则书查询**：（计划中）
- ⚙️ **可配置设置**：启用/禁用各项功能

## 安装方法

### 使用 SillyTavern 内置安装器（推荐）

1. 在 SillyTavern 中打开扩展管理器
2. 点击"安装扩展"
3. 输入仓库 URL：`https://github.com/KuangYang452/ST-TRPG-Tool`
4. 点击"安装"
5. 重启 SillyTavern

### 手动安装

1. 下载或克隆此仓库
2. 重命名文件夹为 `st-trpg-tool`（全小写）
3. 复制到 `public/scripts/extensions/third-party/` 目录
4. 重启 SillyTavern

## 使用说明

安装后，在扩展设置面板中会显示 "TRPG工具箱" 选项：

- **启用骰子功能**：激活骰子投掷界面
- **自动发送骰子结果到聊天**：将骰子结果自动发送到聊天框
- **骰子类型选择**：选择要投掷的骰子类型
- **投骰子按钮**：执行投骰并显示结果

## 系统要求

- SillyTavern 1.10.0 或更高版本
